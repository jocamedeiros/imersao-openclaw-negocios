Estrutura de uma Skill

O mínimo é o SKILL.md. O resto você adiciona conforme a complexidade cresce.

Skill Simples
relatorio-vendas/
Relatório semanal de vendas
relatorio-vendas/
└── SKILL.mdOBRIGATÓRIO
     A receita completa:
     o que faz, quando usar,
     passo a passo e output
Quando usar: automações que dependem só de leitura de dados e geração de texto. O SKILL.md descreve tudo — o agente segue a receita.
Skill Avançada
twitter-banner-creator/
Geração de banners para Meta Ads
twitter-banner-creator/
├── SKILL.mdOBRIGATÓRIO
     A receita principal
├── schema.jsonOPCIONAL
     Input/output tipados e validados
├── examples/OPCIONAL
     Exemplos reais de uso
└── scripts/OPCIONAL
     Scripts de automação (Python, etc.)
Quando usar: automações que envolvem geração de arquivos, APIs externas, múltiplos passos ou lógica complexa. Scripts executam, examples ensinam, schema valida.
Começa com o SKILL.md. Adiciona o resto conforme a skill evolui.
